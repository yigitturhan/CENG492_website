File Signing

Sign any type of file with your e-signature device to ensure authenticity and integrity. Recipients can verify that the file came from you and hasn't been tampered with.

Our file signing feature supports all major file formats and provides cryptographic proof of origin. The signature is embedded directly into the file metadata, making verification seamless across different platforms.

File Encryption

Encrypt your sensitive files using your e-signature device's cryptographic capabilities. Only authorized recipients with the proper keys can decrypt and access your files.

Features end-to-end encryption with perfect forward secrecy. Files are encrypted using hybrid cryptography - symmetric encryption for the data and asymmetric encryption for the keys.

PDF Signing (ETSI-PADES)

Digitally sign PDF documents with full ETSI-PADES compliance. Your signed PDFs can be verified directly in Adobe Acrobat and other PDF readers.

Supports advanced electronic signatures (AdES) with long-term validation. Timestamps and certificate chains are embedded to ensure signatures remain valid over time, even if certificates expire.

Office Document Signing (ETSI-XADES)

Sign Microsoft Office documents (Word, Excel, PowerPoint) with ETSI-XADES standards. Signatures are recognized and verifiable directly within Microsoft Office applications.

Full integration with Microsoft Office ribbon interface. Supports batch signing for multiple documents and maintains formatting integrity during the signing process.

PGP Key Management

Create, manage, and protect your PGP keys using your e-signature device. Seamlessly integrate with existing PGP workflows while keeping your private keys secure in hardware.

Compatible with standard PGP implementations like GnuPG. Your e-signature device acts as a hardware security module, ensuring private keys never leave the device while enabling full PGP functionality.

Secure Gmail Integration

All your Gmail communications are automatically encrypted and signed using your e-signature device or PGP keys. Transparent security without changing your email workflow.

Browser extension seamlessly integrates with Gmail interface. Automatic key exchange with recipients, encrypted draft saving, and visual indicators for message security status.

Authentication API

Passwordless authentication API that allows third-party websites to authenticate users through their e-signature devices. No more passwords, just cryptographic proof of identity.

HTTP/S Authorization

Header-based authorization mechanism for restricted URLs. Access protected resources using cryptographic signatures in HTTP headers instead of traditional session cookies.

Implements custom HTTP authentication scheme with replay protection. Perfect for API access, administrative interfaces, and any web resource requiring strong authentication.

Credential Management

Secure vault for storing and managing your credentials, protected by your e-signature device. Access your passwords, API keys, and certificates from any computer with your device.

[Credential Vault Interface]

Encrypted credential storage with automatic form filling. Supports secure sharing of credentials within teams, audit logging, and automatic credential rotation for supported services.
